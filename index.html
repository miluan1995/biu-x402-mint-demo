BIU

BIU 是 BSC 上的 x402 fair mint 代币。网页现在不再做钱包连接，只展示代币、Mint Skill 安装方式和实时 mint 数据。玩家让自己的 Agent 安装 Skill 后，用 USDC 走 quote / dry-run / 确认 mint 流程。

Mint Skill 安装

给玩家 Agent 的安装说明

复制/安装这个 Skill，让玩家 Agent 输入“mint 1 份 BIU”时，先生成报价和 dry-run，不会直接扣 USDC。

agent-skill/bui-mint/SKILL.md

验证安装：

cd bsc-x402-launchpad
node scripts/bui-mint.js agent-context --json --no-input
./agent-skill/bui-mint/demo.sh

玩家 Agent 常用命令：

node scripts/bui-mint.js quote --payer <玩家钱包> --lots 1 --json --no-input
node scripts/bui-mint.js dry-run --payer <玩家钱包> --lots 1 --json --no-input
